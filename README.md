# 🍕 Pizza Sales Analysis Dashboard | Power BI Project

## 📌 Project Overview

This project focuses on analyzing pizza sales data and building an interactive **Pizza Sales Analytics Dashboard using Microsoft Power BI**.

The objective of this dashboard is to help businesses understand sales performance, customer ordering patterns, product popularity, and revenue trends through data-driven insights.

The dashboard provides a centralized view of key performance indicators (KPIs), sales trends, pizza category analysis, size preferences, and top & bottom-performing products.

---

## 🎯 Business Problem Statement

The pizza business wants to analyze sales performance and understand:

- Overall revenue and order performance
- Customer purchasing trends
- Peak ordering periods
- Popular pizza categories and sizes
- Best-selling and low-performing pizzas
- Product contribution towards total sales

The dashboard helps management make informed decisions to improve sales strategies, inventory planning, and customer satisfaction.

---

# 📊 KPI Requirements

The dashboard tracks the following key metrics:

### 1. Total Revenue
- Calculates the total revenue generated from all pizza orders.

### 2. Average Order Value
- Measures the average amount spent per customer order.

**Formula:**

Average Order Value = Total Revenue / Total Orders


### 3. Total Pizzas Sold
- Displays the total quantity of pizzas sold.

### 4. Total Orders
- Shows the total number of customer orders.

### 5. Average Pizzas Per Order
- Calculates the average number of pizzas purchased per order.

**Formula:**

Average Pizzas Per Order = Total Pizzas Sold / Total Orders
